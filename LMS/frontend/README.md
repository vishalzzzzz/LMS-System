# Smart Library Frontend

React + Vite frontend for the Smart Library Borrowing System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Modern React with Hooks
- React Router for navigation
- Tailwind CSS for styling
- Axios for API calls
- Context API for state management
- Hot toast notifications
- Responsive design

## Pages

- **Login/Signup**: Authentication
- **Dashboard**: Overview and quick actions
- **Books**: Browse and borrow books
- **Active Borrows**: Manage current borrows
- **History**: View past borrows
- **Payments**: Track payment status
- **Profile**: User information

## Components

- **Navbar**: Navigation menu
- **BorrowModal**: Book borrowing dialog
- **ReturnModal**: Book return dialog
- **PrivateRoute**: Protected route wrapper
